# OttomanOCR-search
This repository includes the OCR'd output for 28 Ottoman Turkish printed documents..

The data is processed with the OCR model for printed Ottoman Turkish [Ottoman-Print-OCR-Qwen4B](https://huggingface.co/enesyila/Ottoman-Print-OCR-Qwen4B). The regions and lines were detected with two custom YOLO26s models (Follow the links for the [region-detection model](https://github.com/Koras0ff/OttomanOCR-search/blob/main/yolo26s_region_printed_24_06_2026.pt) and the [line-detection model](https://huggingface.co/enesyila/yolo26s_ottoman_printed_line)).
